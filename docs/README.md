# gateway-oss documentation

This directory holds public-facing Gateway documentation, authored directly into this repository.

Planned contents (added as milestones land):

- User guide for the open-source Gateway components.
- Public API references for the SDKs and CLI.
- Plug-in authoring guide for agent-platform and attestation plug-ins.
- Example walkthroughs paired with `../examples/`.

At MVP this directory is intentionally minimal. Substantive public documentation lands alongside the components it documents.
